This website is created using HTML 5, CSS 3, and JavaScript. This page consists of HTML5 Web Audio Players, and custom players from EC3 (Radio Streaming Software).


Try it out!

Players are hidden by default to prevent overlapping on the site. Click one of the buttons to reveal either of the players along with a load stream URL form, it may be defaulted to Fuse/Blast radio temp. until the form is updated.
